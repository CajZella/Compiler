2. 判断x和y是否相邻 -> 邻接矩阵

计数器：
1. 每个传送有关的结点都有一个计数器，记录该结点涉及的传送指令的条数
2. 所有节点都有记录图中当前与它相邻的结点个数的计数器

其他信息记录：
1. 低度数的传送无关的结点simplify worklist
